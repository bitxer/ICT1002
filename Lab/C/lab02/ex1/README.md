# Exercise 1: Array Expressions
## Task Description
Suppose that the following declarations and assignments have been made in a C program:
```
int a[4]={-1,2,10,7};
int b[4];
for(int i=0;i<4;i++)
    b[i]=a[3 - i];
```
## Requirements
Write a program to print the values for each of the following expressions.
1. a[3]
2. b[3]
3. b[a[1]]
